Задание 1
Создайте новый проект веб-приложения ASP.NET Core,
построенный на базе шаблона ASP.NET Core Empty.
Задание 2
Подключите к проекту (в классе “Program”) сервисы,
необходимые для работы с контроллерами и представлениями
(MVC сервисы).
Задание 3
Настройте сопоставление маршрутов с контроллерами
по-умолчанию. Для этого используйте метод 
“MapControllerRoute” и шаблон (“pattern”) маршрута
“{controller=Home}/{action=Index}/{id?}”.
Задание 4
Создайте контроллер для главной страницы вебприложения (например, “Home”).
В созданном контроллере главной (домашней) страницы
определите действие “Index”.
Действие (“action”) “Index” должно возвращать строку с
любым текстовым сообщением (например, “Hello, world!”).
Помните, что контроллеры следует размещать в
директории “Controllers”.
Задание 5
Создайте сервис, отвечающий за генерацию приветствия
в зависимости от текущего времени суток (например,
“WelcomeMessageService” или “GreetingService”.
Сервис должен содержать метод для получения
приветственного сообщения (например, “GetMessage” или
“GetWelcomeMessage”).
Примеры возможных сообщений:
1. Доброе утро!
2. Добрый день!
3. Добрый вечер!
4. Доброй ночи!
   Задание 6
В контроллере главной (домашней) страницы
“HomeController” определите действие “Welcome”, которое
будет возвращать приветственное сообщение, в зависимости
от текущего времени.
Контроллер должен использовать сервис, созданный в
предыдущем задании.
Действие “Welcome” в контроллере “Home” должно быть
доступно только для HTTP GET запросов.
Задание 7
В контроллере главной (домашней) страницы определите
действие “Time”, которое должно возвращать текущее время
(в любом формате: либо только время, либо только дата, либо
дата и время).
